# chemin-le-plus-court-plugin-qgis
This python plugin allows to select the linear objects forming the shortest path between two selected linear objects. 

## Pré-requis

qgisMinimumVersion=3.4

## Fonctionnalités

French :
* A partir de deux objets linéaires sélectionnés, calcul et sélection des tronçons formant le plus court chenmin entre les deux tronçons d'origine.


English : 
* From two selected linear objets, makes a selection of the lineair objets forming the shortest path.
